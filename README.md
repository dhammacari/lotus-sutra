# lotus-sutra

# 🪷 The Gist of Lotus Sutra: Chapters 2 & 16

โปรเจกต์รวบรวมแก่นธรรมจาก **สัทธรรมปุณฑรีกสูตร (Lotus Sutra)** บทที่ 2 และ 16 ตามแนวทางของ **นิกายนิชิเรนชู (Nichiren Shu)** โดยนำเสนอในรูปแบบ Minimalist Web Page เพื่อการศึกษาและการเจริญสติ

## 📖 Concept
โปรเจกต์นี้มุ่งเน้นการถ่ายทอด "ความลับโดยพยัญชนะ" ของพระสูตรผ่านอุปมาแห่ง **บัวขาว (Pundarika)** ที่เบ่งบานจากสัทธรรมและสัทธาอันลึกซึ้งในตถาคตโพธิ แม้จะต้องเผชิญกับโคลนตมหรืออุปสรรคแห่งสุญญตาธรรมก็ตาม

## 🛠 Features
- **Artisan Design:** เน้นความเรียบง่าย สบายตา และรองรับ Dark Mode 100%
- **Bilingual Content:** สรุปใจความสำคัญทั้งภาษาไทยและภาษาอังกฤษ
- **Direct Gist:** - **Chapter 2 (Hoben-pon):** กุศโลบายและเอกยาน
  - **Chapter 16 (Juryo-hon):** อายุขัยอันเป็นนิรันดร์ของพระตถาคต
